# Source Selection Memo

This is a source-selection step only. No case abstraction has been performed, no source text has been copied, and no patient-identifying information has been added.

## How Sources Were Identified

Sources were identified through targeted public web search focused on open educational case reports and PubMed Central pages. The search prioritized sources with public metadata, visible titles, and enough structure to suggest an initial presentation plus downstream assessment or plan.

The selection process used metadata-level review only. It did not scrape websites, copy case text, copy paywalled text, or reproduce source passages.

## Most Promising Source Types

The most promising sources were open-access case reports with:

- a clear presenting concern;
- a downstream specialist assessment, pathology result, imaging-based interpretation, or final plan;
- explicit diagnostic uncertainty;
- a shift from initial suspected diagnosis to later assessment;
- enough public metadata to support citation-only indexing.

Strong candidate domains included oral medicine, endodontics, periodontics, orthodontics, cardiology, neurology, gastroenterology, and dermatology.

## Unsuitable Source Types

Less suitable sources included:

- paywalled-only case reports;
- sources whose educational value depends on copied images or tables;
- broad reviews without a discrete case transition;
- cases with too much identifying detail;
- therapy reports without a clear initial-to-downstream cognitive transition;
- cases requiring the Dataset v0.2 team to generate clinical advice.

## Feasibility of 10 Cases

Ten cases appear feasible for Dataset v0.2 abstraction. The current shortlist contains 10 candidates with enough apparent structure to support paraphrased educational artifact pairs.

This feasibility judgment is provisional. A second reviewer should verify eligibility before abstraction begins.

## Next Step

The next step should be abstraction, but only after:

1. a second reviewer verifies the 10 shortlisted sources;
2. copyright and privacy risk labels are confirmed;
3. any source requiring full-text copying, image reproduction, or identifiable detail is removed;
4. each selected case is assigned an abstraction plan.

## Copyright and Privacy Concerns

Main copyright concern: even open-access case reports should not be copied into Dataset v0.2. The dataset should store citation-only metadata and original paraphrased educational abstractions.

Main privacy concern: case reports often include age, condition rarity, timeline, institution, imaging, or treatment history. Abstractions should generalize or omit details that are not necessary for studying the cognitive transition.

## Second Reviewer Recommendation

A second reviewer should independently verify eligibility before abstraction. The reviewer should check:

- public accessibility;
- copyright risk;
- privacy risk;
- presence of an initial artifact and downstream artifact;
- whether the cognitive transition can be abstracted without copying source text;
- whether the case should remain in the shortlist.
