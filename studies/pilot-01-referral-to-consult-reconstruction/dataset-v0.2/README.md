# Dataset v0.2

Dataset v0.2 is intended to test the referral-to-consult reconstruction workflow on messier published educational cases or citation-linked educational case materials.

Dataset v0.2 is not yet a clinical dataset. It should not include real identifiable patient information, should not copy copyrighted case reports, and should not be used for clinical guidance or patient-care decisions.

Acceptable materials include:

- citation-linked case summaries;
- paraphrased educational abstractions;
- synthetic-derived pairs based on public educational patterns;
- public educational cases abstracted without copying source text.

The purpose is methodology testing only: to examine whether the reconstruction protocol, coding framework, and rubric remain usable when moving beyond clean synthetic cases.
