# Source-Distance Checklist

Use this checklist to decide whether each abstraction is sufficiently distant from the source report. Do not copy or quote source text while completing this checklist.

Reviewer decision options:

- pass
- pass with caution
- revise before coding
- reject for this dataset

## ABS-001

- [ ] Does the abstraction avoid copied source sentences?
- [ ] Does it avoid unique source phrasing?
- [ ] Does it avoid reproducing the source narrative sequence too closely?
- [ ] Does it avoid source-specific timeline detail?
- [ ] Does it avoid institution / geography details?
- [ ] Does it avoid exact demographics unless methodologically necessary?
- [ ] Does it preserve only the cognitive transition needed for the study?
- [ ] Does it feel like a new educational abstraction rather than a compressed case report?
