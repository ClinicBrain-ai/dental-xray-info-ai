# Non-Guidance Checklist

Use this checklist to ensure each abstraction remains a methodology artifact rather than clinical advice.

Reviewer decision options:

- pass
- pass with caution
- revise language
- reject for non-guidance concerns

## ABS-001

Check whether it:

- [ ] avoids telling clinicians what to do
- [ ] avoids treatment recommendations as advice
- [ ] avoids patient-care instructions
- [ ] avoids clinical validation claims
- [ ] avoids implying the abstraction is real documentation
- [ ] clearly states educational / methodology-only use
- [ ] uses cognitive-transition language rather than clinical directive language
