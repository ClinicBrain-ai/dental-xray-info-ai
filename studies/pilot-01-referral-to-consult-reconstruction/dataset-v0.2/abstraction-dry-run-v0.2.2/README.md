# Dataset v0.2.2 Limited Abstraction Dry Run

This folder contains a 2-case abstraction dry run for Dataset v0.2.1. It is not the full Dataset v0.2 abstraction step.

The outputs are paraphrased educational abstractions for methodology testing only. They are not real clinical documentation, are not clinical guidance, and should not be used for patient care.

No source narrative text should be copied into this folder. The dry run exists to test whether the abstraction protocol, output template, and safeguards can produce usable reconstruction artifacts without preserving patient-identifying information or copyrighted source language.
