# Dataset v0.2 Future Work Notes

This file records possible future directions only. It is not a roadmap and does not authorize any work.

## Additional Controlled Candidate Testing

Status: not currently authorized.

Additional controlled candidate testing may be useful only after a new gate decision and renewed source-distance, privacy, copyright, and methodological review.

## Compression vs Uncertainty-Loss Coding Refinement

Status: not currently authorized.

Future refinement may clarify when compressed cognition becomes uncertainty loss. Any refinement should be documented before additional coding is attempted.

## Second-Coder Studies

Status: not currently authorized.

Second-coder work may help assess whether the current coding framework is interpretable by more than one reviewer. It would require a separate workflow and explicit authorization.

## Published Educational Case Comparisons

Status: not currently authorized.

Future comparisons across published educational cases may be methodologically useful, but only if they preserve copyright distance, privacy safeguards, and non-guidance boundaries.
