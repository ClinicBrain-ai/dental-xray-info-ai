# Candidate Source Index

Dataset v0.2 candidate source selection. This file records citation and metadata only. It does not copy case text, does not abstract cases, and does not include patient-identifying information.

Eligibility status values:

- include: strong candidate for Dataset v0.2 abstraction.
- maybe: possible candidate, but requires additional review before abstraction.
- exclude: not suitable for Dataset v0.2 without a major change in scope or permissions.
