# Case Coding: P1-V01-003

## Case ID

P1-V01-003

## Source Artifacts Used

Referral Letter and Specialist Consultation Note only.

## Reconstructed Source Cognition

The referring dentist suspected a failed root-treated tooth but recognized that vertical root fracture, occlusal trauma, and endodontic pathology could produce similar symptoms. Referral sought prognosis and treatment-path selection.

## Preserved Cognition

- Previously root-treated premolar remained the focus.
- Fracture versus endodontic failure remained visible.
- Retreatment versus extraction remained the management fork.

## Transformed Cognition

The specialist elevated vertical root fracture from possibility to leading concern and added a staged imaging/probing plan.

## Missing Cognition

Referral lacks periodontal probing data and detailed restoration/fracture risk information.

## Newly Introduced Cognition

Localized deep probing and CBCT as decision tools are introduced downstream.

## Drift Characterization

Diagnostic reframing and mechanism substitution toward structural root failure. Drift is moderate and partially explained by specialist reasoning.

## Uncertainty Assessment

Moderate-high confidence in source cognition. The consultation provides enough rationale to reconstruct the main uncertainty but not all evidence behind the original suspicion.
